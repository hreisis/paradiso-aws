# paradiso-aws
A MERN stack web app for a boba tea shop which allows customers and staff to post and update information.

## Built With
* React
* Bootstrap
* Node.js
* Express.js
* MongoDB
* Mongoose
* AWS EC2

## Site Link
http://www.paradiso-tea.com
